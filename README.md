# Big Project

## 설치 방법

### 1. 프론트엔드
```
cd frontend
npm install
```

### 2. 문서 처리 서비스
```
cd backend/document-api
npm install
```

### 3. AI API
```
cd backend/ai-api
python -m venv venv
venv/Scripts/activate
pip install -r requirements.txt
```

### 4. Spring Boot
IntelliJ에서 backend/core-api 폴더 열면 자동 세팅
